Senzing SDK in my development environment

This exercise requires that you already have a Go development environment installed on your computer.

Download and run any of the following files:

  1. senzing_hello_world/main.go - A simple test of connectivity to Senzing engine.
  2. senzing_load_truthsets/main.go - Load and query the Senzing truth-set-data.
  3. senzing_load_user_data/main.go - Load custom data.
  4. senzing_method_help/main.go - Show how to display help.

Using the files as examples, build your own Go application using the Senzing SDK.
